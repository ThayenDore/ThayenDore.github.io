SFML Snake

This was a solo project developed in C++ using SFML to create windows and render it.

It consists of 5 levels using the base concept of snake and developing further machanics from it, using new organisations of obstacles to enhance difficulty.

I created all of the game logic code, and created the framework that uses SFML to render the sprites.

The snake uses a linked list to connect all of it's segments and to simply remove them.
