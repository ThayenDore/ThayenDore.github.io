Astrift Convergent

This was a group project done in collaboration with other courses, created in unity using C#. Designed as a wave based top down shoot-em-up with 3 types of enemies with variations on them and a final boss at the end of 5 waves of a set time duration.

I was responsible for programming the logic for the enemies and a large portion of the player logic, as well as making it usable for the designers using the in engine inspector.

I was also resposible for creating the system that organised the enemies into wave, and the logic behind the scrolling backgrounds.

The game also uses an upgrade system that asks the player to choose between 2 different upgrades on each wave
